# terminal_game
Little game to run in terminal.

---

## Branches

- Feature/add_monster:
  Monster that chases Player in through the map. 

---

## Build

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

---

## Movement

WASD movement

- W - up
- A - left
- S - down
- D - right
- Q - quit
