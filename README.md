# terminal_game
2D-game to run in terminal with a player that moves on a map.

---

## Branches

- Feature/add_monster:
  Monster that chases Player in through the map. 

---

## Build

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

---

## Movement

WASD movement

- W - up
- A - left
- S - down
- D - right
- Q - quit
